# Short-form Fatigue Experiment Platform

A web-based experiment platform designed to collect and analyze user responses to short-form video content.
This project focuses on measuring short-form media fatigue by combining behavioral interaction logs with survey-based responses.

## Overview

This project was developed to study how users perceive and respond to short-form video content, especially in relation to fatigue, information overload, content reliability, negative content, privacy concerns, FoMO, and AI-generated content.

The system is designed to collect both explicit survey responses and behavioral data such as video viewing actions, likes, swipes, pauses, and session activity.
By connecting user behavior data with survey results, the project aims to support a more data-driven analysis of short-form media fatigue.

## Key Features

* Short-form video experiment interface
* Survey-based fatigue factor measurement
* Behavioral event logging during video interaction
* Session-based user activity tracking
* Data collection structure for combining survey and behavior logs
* Admin/data management flow for research analysis

## Data Flow

The project follows a data collection and analysis pipeline:

1. Users participate in a short-form video viewing experiment
2. Survey responses and behavioral events are collected
3. Interaction logs are stored by session and participant
4. Collected data can be used for fatigue factor analysis and user behavior interpretation

## Tech Stack

* Frontend: React, TypeScript, Vite
* UI: Tailwind CSS, shadcn/ui
* Backend: API-based event collection structure
* Database: Supabase
* Tools: Git, GitHub

## Project Purpose

The purpose of this project is to move beyond survey-only research by combining self-reported user perception with actual behavioral logs.
This approach helps analyze how digital content and platform interactions influence user fatigue and decision-making behavior.

## Relevance

This project demonstrates experience in designing a data collection pipeline, structuring user interaction logs, and connecting digital behavior data to research-based analysis.
It is related to data engineering, user behavior analytics, human-AI interaction, and data-driven decision support.
